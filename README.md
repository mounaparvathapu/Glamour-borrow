# Glamour Borrow – Online Fashion Rental Platform

A web-based fashion rental platform that allows users to browse, rent, and return stylish dresses and fashion accessories for special occasions. The platform promotes sustainable fashion by making designer outfits accessible without the need for purchasing them.


---

## 🧰 Tech Stack

### Frontend

- HTML5
- CSS3
- JavaScript

### Design & Development

- Human-Computer Interaction (HCI) Principles
- User-Centered Design
- Responsive Web Design

---

## 🎯 Project Objective

The objective of Glamour Borrow is to provide an affordable and sustainable alternative to purchasing expensive fashion items. Users can rent outfits for weddings, parties, festivals, and special occasions, then return them after use. This approach helps reduce fashion waste while giving customers access to a wide variety of stylish outfits at a lower cost.
